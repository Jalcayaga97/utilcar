# Utilcar Sanity Studio

Schemas alineados con `utilcar-web/src/lib/sanity/queries.js`.

## Inicio rápido

```bash
npm install
cp .env.example .env   # editar PROJECT_ID
npm run dev
```

## Seed

```bash
SANITY_PROJECT_ID=xxx SANITY_API_TOKEN=skxxx npm run seed
```

Ver `utilcar-web/docs/SANITY_INTEGRACION.md` para la checklist completa.
